# ✅ Чек-лист настройки автоматической загрузки

## Быстрая проверка (3 минуты)

Пройдитесь по всем пунктам для настройки авто-загрузки в GitHub.

### □ Шаг 1: GitHub Personal Access Token

- [ ] Перейти на https://github.com/settings/tokens
- [ ] Нажать **Generate new token (classic)**
- [ ] Ввести название: `BotBuilder Auto-Deploy`
- [ ] Выбрать scope: **✅ repo**
- [ ] Нажать **Generate token**
- [ ] **Скопировать токен** (покажется один раз!)

### □ Шаг 2: Секрет в GitHub Repository

- [ ] Открыть https://github.com/kkav45/coopbot
- [ ] Перейти в **Settings** → **Secrets and variables** → **Actions**
- [ ] Нажать **New repository secret**
- [ ] Создать секрет:
  - Name: `GH_PAT`
  - Value: ваш токен из шага 1
- [ ] Нажать **Add secret**

### □ Шаг 3: Подключение в конструкторе

- [ ] Открыть `index.html` в браузере
- [ ] Перейти в **⚙️ Настройки**
- [ ] Заполнить:
  - [ ] GitHub Token: ваш токен
  - [ ] Repository: `coopbot`
  - [ ] Username: `kkav45`
- [ ] Нажать **Подключить**
- [ ] Проверить статус: должно быть "GitHub подключён"

### □ Шаг 4: Включение авто-синхронизации

- [ ] В **Настройки** → **Авто-синхронизация**
- [ ] Включить переключатель
- [ ] Проверить: индикатор горит зелёным

### □ Шаг 5: Проверка работы

- [ ] Создать тестового бота:
  - [ ] Нажать **➕ Создать бота**
  - [ ] Ввести название: `Тестовый бот`
  - [ ] Username: `test_bot`
  - [ ] Токен: любой (для теста)
  - [ ] Нажать **Создать бота**
- [ ] Нажать **🚀 Опубликовать**
- [ ] Открыть https://github.com/kkav45/coopbot/commits/main
- [ ] Проверить: появился новый коммит "🤖 Auto-commit bot updates"

## 🎉 Готово!

Если все пункты выполнены — автоматическая загрузка настроена!

## 🔍 Проверка статуса

### В конструкторе

- Индикатор в sidebar должен быть **зелёным**
- Текст: "GitHub подключён"

### В GitHub

1. Откройте https://github.com/kkav45/coopbot/actions
2. Выберите workflow **Auto-Commit Bot Changes**
3. Все запуски должны быть ✅ (зелёные)

### В репозитории

1. Откройте https://github.com/kkav45/coopbot
2. Проверьте папку `bots/`
3. Должна появиться папка с вашим ботом

## 🐛 Если что-то пошло не так

### Ошибка "GitHub не подключён"

- [ ] Проверьте токен в настройках
- [ ] Убедитесь, что у токена scope `repo`
- [ ] Проверьте, что репозиторий `coopbot` существует

### Workflow не запускается

- [ ] Откройте вкладку **Actions** в GitHub
- [ ] Убедитесь, что Actions включены
- [ ] Проверьте наличие секрета `GH_PAT`

### Ошибка аутентификации

- [ ] Создайте новый Personal Access Token
- [ ] Обновите секрет `GH_PAT` в GitHub
- [ ] Переподключитесь в конструкторе

### Бот не загружается

- [ ] Откройте консоль браузера (F12)
- [ ] Проверьте на ошибки (красный текст)
- [ ] Попробуйте нажать **🚀 Опубликовать** ещё раз
