## 👥 Роли ботов

### 1. 💰 Бот-Бухгалтер

**Назначение:** Финансовый учёт, отчётность, анализ финансового состояния.

**Функции:**
- 📊 Учёт доходов и расходов
- 📈 Финансовые отчёты (P&L, Cash Flow, Balance Sheet)
- 💵 Расчёт налогов и зарплат
- 🔍 Анализ финансовых показателей
- ⚠️ Предупреждения о кассовых разрывах
- 📑 Работа с первичной документацией
- 🏦 Интеграция с банками и платёжными системами

**Сценарии:**
```
/finance_report      — Отчёт за период
/cashflow           — Движение денег
/profit_loss        — Отчёт о прибылях и убытках
/tax_calc           — Расчёт налогов
/invoice_create     — Создание счёта
/expense_add        — Добавление расхода
/budget_plan        — Планирование бюджета
```

**База данных:**
- Таблица `Transactions` (транзакции)
- Таблица `Invoices` (счета)
- Таблица `Contracts` (контракты)
- Таблица `TaxRates` (налоги)
- Таблица `FinancialReports` (отчёты)

---

### 2. 🛒 Бот-Продавец-Консультант

**Назначение:** Продажи, консультации, работа с клиентами.

**Функции:**
- 📦 Управление каталогом товаров/услуг
- 💬 Консультации клиентов
- 🎯 Рекомендации товаров
- 🛒 Оформление заказов
- 📞 Обработка возражений
- ⭐ Сбор отзывов
- 🔔 Напоминания о брошенных корзинах
- 🎁 Программа лояльности

**Сценарии:**
```
/catalog            — Показать каталог
/product/{id}       — Информация о товаре
/recommend          — Подбор товаров
/cart               — Корзина
/order_create       — Оформление заказа
/order_status       — Статус заказа
/consultation       — Консультация
/promo              — Акции и скидки
```

**База данных:**
- Таблица `Products` (товары)
- Таблица `Categories` (категории)
- Таблица `Customers` (клиенты)
- Таблица `Orders` (заказы)
- Таблица `Reviews` (отзывы)
- Таблица `Consultations` (консультации)

### 3. 🎯 Бот-Топ-Менеджер

**Назначение:** Стратегическое планирование, аналитика, мониторинг рынка.

**Функции:**
- 📰 Мониторинг новостей и трендов
- 📊 Анализ рынка и конкурентов
- 🎯 Стратегическое планирование
- 💡 Генерация идей и рекомендаций
- 📈 Прогнозирование трендов
- 🔍 SWOT-анализ
- 📋 Подготовка отчётов для руководства
- 🌐 Мониторинг соцсетей и отзывов

**Сценарии:**
```
/market_review      — Обзор рынка
/competitors        — Анализ конкурентов
/trends             — Тренды отрасли
/strategy           — Стратегические рекомендации
/swot               — SWOT-анализ
/forecast           — Прогноз
/news_digest        — Дайджест новостей
/kpi_report         — Отчёт по KPI
```

**База данных:**
- Таблица `MarketData` (данные рынка)
- Таблица `Competitors` (конкуренты)
- Таблица `Trends` (тренды)
- Таблица `Strategies` (стратегии)
- Таблица `KPIs` (показатели)
- Таблица `NewsSources` (источники новостей)

**Интеграции:**
- RSS ленты новостей
- API социальных сетей
- Google News API
- SimilarWeb API (конкуренты)
- Яндекс.Wordstat (тренды)

**Назначение:** Критическое мышление, поиск альтернатив, оптимизация решений.

**Функции:**
- 🔄 Генерация альтернативных решений

- 🔍 Анализ "что если"

- 🚩 Выявление узких мест
- 🎲 Мозговой штурм

**Сценарии:**
```
/alternatives       — Альтернативные решения
/critique           — Критика предложения
/risks              — Анализ рисков
/optimize           — Оптимизация процесса
/what_if            — Сценарий "что если"
/compare            — Сравнение вариантов
```

**База данных:**
- Таблица `Decisions` (решения)
- Таблица `Alternatives` (альтернативы)
- Таблица `Risks` (риски)
- Таблица `Optimizations` (оптимизации)
- Таблица `Debates` (дебаты/обсуждения)

## 🔗 Межботовая коммуникация

### Протокол обмена сообщениями

```json
{
  "message_id": "msg_123456",
  "timestamp": "2026-03-01T10:00:00Z",
  "from_bot": "seller_bot",
  "to_bot": "accountant_bot",
  "type": "request|response|notification|broadcast",
  "priority": "low|normal|high|urgent",
  "topic": "finance|sales|strategy|operations",
  "content": {
    "action": "create_invoice",
    "data": {
      "order_id": "ORD-123",
      "amount": 10000,
      "customer": "Client LLC"
    }
  },
  "context": {
    "conversation_id": "conv_789",
    "user_id": "user_456",
    "thread_id": "thread_321"
  }
}
```

### Сценарии взаимодействия

#### 1. Продажа → Бухгалтерия
```
Продавец: "Заказ оформлен, нужен счёт"
   ↓
Бухгалтер: "Счёт создан, проводка выполнена"
```

#### 2. Топ-менеджер → Все
```
Топ-менеджер: "Вопрос: как увеличить продажи на 20%?"
   ↓
Продавец: "Предлагаю акции и кросс-продажи"
Бухгалтер: "Нужен бюджет на маркетинг"
Альтернативщик: "А может снизить издержки вместо роста продаж?"
```

#### 3. Дебаты: Топ-менеджер ↔ Альтернативщик
```
Топ-менеджер: "Предлагаю выйти на новый рынок"
   ↓
Альтернативщик: "Риски: конкуренция 80%, затраты 5М, срок окупаемости 2 года"
   ↓
Топ-менеджер: "Но потенциал рынка 50М в год"
   ↓
Альтернативщик: "Альтернатива: усилить позиции на текущем рынке"
```

## 💡 Примеры использования

### Сценарий 1: Оформление продажи

```
1. Клиент → Продавец: "Хочу купить товар X"
2. Продавец: "Оформляю заказ... Заказ ORD-123 создан"
3. Продавец → Бухгалтер: message_bus.send('create_invoice', order_id='ORD-123')
4. Бухгалтер: "Счёт INV-456 создан, проводка выполнена"
5. Продавец → Клиент: "Оплата по счёту INV-456"
6. Клиент оплачивает
7. Продавец → Бухгалтер: "Оплата получена"
8. Бухгалтер: "Проводка: доход 10000₽"
9. Продавец → Топ-менеджер: "Продажа совершена"
10. Топ-менеджер: "KPI обновлён: +10000₽ к плану"
```

### Сценарий 2: Стратегическая сессия

```
1. Пользователь → Топ-менеджер: "Как увеличить прибыль?"
2. Топ-менеджер: "Анализирую... Предлагаю: 1) Расширить ассортимент 2) Выйти на новый рынок"
3. Пользователь → Альтернативщик: "Что думаешь?"
4. Альтернативщик: "Риски: 
   - Расширение: затраты 500К, риск 40%
   - Новый рынок: затраты 2М, риск 70%
   Альтернатива: оптимизировать издержки (экономия 300К/мес)"
5. Пользователь → Бухгалтер: "Финансовый прогноз?"
6. Бухгалтер: "Расширение: ROI 25%, срок 12 мес
   Новый рынок: ROI 40%, срок 24 мес
   Оптимизация: ROI 100%, срок 3 мес"
7. Топ-менеджер ↔ Альтернативщик: [дебаты]
8. Итог: Решение принято с учётом всех мнений
```
